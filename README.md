# ConroleFinanceiro

Este repositório contém um projeto de controle financeiro desenvolvido para fins de estudo e prática.  
O objetivo é ajudar a fixar conteúdos relacionados a backend, organização de dados, lógica de cálculo e registro de transações.

## Objetivo do projeto

O sistema permite gerenciar movimentações financeiras básicas, como entradas e saídas.  
Serve como base para explorar conceitos de:

- Estrutura de API ou aplicação backend
- Persistência de dados
- Organização de entidades financeiras
- Tratamento de registros e cálculos simples
- Separação de responsabilidades no código

Este projeto não é voltado para produção.  
Seu propósito principal é estudo, testes e aprimoramento de conhecimento.

## Tecnologias utilizadas

- .NET / C#
- Entity Framework Core
- Banco de dados relacional
- API REST ou aplicação console/interface simples
